# STE Writing Coach Plugin

A starter project for a Custom GPT that checks and rewrites text using an ASD-STE100-oriented workflow.

> This is an independent writing aid. It does not certify ASD-STE100 compliance and is not endorsed by ASD or STEMG.

## Planned components

- FastAPI API for deterministic checks
- OpenAPI schema for a Custom GPT Action
- Strict technical and STE-inspired modes
- Terminology support
- Rewrite comparison safeguards
- Docker deployment files
- Automated tests

## Repository

This repository is being initialized from the prepared STE Writing Coach project.
