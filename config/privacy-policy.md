# Privacy policy template

Replace this template with the deployed service operator's legal details before publishing the GPT.

## Data processed
The STE Writing Coach API receives text that a user submits for analysis, rewriting, or comparison. It can also receive user-supplied terminology.

## Purpose
The service processes the submitted text only to return controlled-language findings and rewrite candidates.

## Storage
The reference implementation does not store request bodies in a database. Hosting providers, reverse proxies, and application logs can still retain metadata or request content depending on deployment settings. Disable body logging and configure retention before production use.

## Model training
The reference API does not train a model on submitted text.

## Sharing
Do not send confidential, export-controlled, classified, personal, medical, or safety-critical information to a deployment unless the organization has approved the hosting, access controls, and retention policy.

## Contact
Add the service operator's name, postal address, privacy contact, and applicable legal basis here.
